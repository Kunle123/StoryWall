# UI mockups (static HTML)

Open any `.html` file in a browser (double-click or “Open with Live Server”).

| File | Purpose |
|------|---------|
| `feature-page-timeline-summary.html` | Featured creator spotlight + example timeline cards with blurbs |
| `home-multiple-stories.html` | **Home-style grid**: many stories at once, visitor scans summaries before opening one |

### Product notes

- **Home** should optimize for **scanning**: many cards, consistent summary length, optional topic chips + search.
- **Summary** can stay as trimmed `description` at first; later add a dedicated **`summary`** (or `deck`) field for 1–2 sentence copy distinct from the long timeline description.
